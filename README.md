# Keypress and Attention

A web-based keypress game designed to **test reaction time and attention**, implementing concepts from **Human-Computer Interaction (HCI)**.

---

## 🎮 Game Description

Keypress challenges the user to respond quickly and accurately to randomly generated letters and numbers:

- **Letters (A-Z)** → Press **A**  
- **Numbers (0-9)** → Press **L**  
- Each game consists of **20 random characters**.  
- After each keypress, the game shows:
  - Correct Moves
  - Wrong Moves
  - Time taken for the keypress (seconds)
- At the end of the game, it displays:
  - Total Correct Moves
  - Total Wrong Moves
  - Total Time
  - Average Time per Keypress

---

## 📝 Instructions

1. Click **Start Game** on the Home screen.  
2. Watch the character in the box.  
3. Press **A** for letters, **L** for numbers.  
4. Try to **respond as quickly and accurately** as possible.  
5. You can **quit the game** anytime to see your results.  
6. After the game ends, you can **Retry** or go **Home**.

## 📌 Features

- Randomized characters per game  
- Per-keypress timing measurement  
- Real-time feedback (Correct/Wrong + Time)  
- End-of-game summary with average response time  
- Retry and Home functionality
